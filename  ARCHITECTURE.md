## System Overview

AfyaDirect is a comprehensive telemedicine platform designed for East Africa, connecting patients with healthcare providers through a multi-platform application. The system supports real-time consultations, secure payments, medical record management, and administrative oversight.

### Key Metrics
- **Expected Users**: 100,000+ patients, 5,000+ doctors
- **Concurrent Users**: 10,000+
- **Response Time**: < 2 seconds
- **Availability**: 99.9% uptime
